# Tenant Hub Frontend (Vite + React)

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Create env file from [.env.example](.env.example):

```bash
cp .env.example .env
```

3. Start dev server:

```bash
npm run dev -- --port 8888
```

## Build and test

```bash
npm run lint
npm run test
npm run build
```
